# yoo-image-zoom

## Installation setup

```
npm install yoo-image-zoom --save

```

## Props

| Name | Type   | Desc                          |
| ---- | ------ | ----------------------------- |
| src  | string | 图片路径                      |
| alt  | string | 图片 alt 属性                 |
| size | object | 移动框大小，默认为 150 \* 150 |

## Tips

放大比例=容器宽度/移动框宽度
